# nyc_congestion_pricing

Code and data for NYC Congestion Pricing policy evaluation paper

-   **Collaborators**:  Tim Fraser, Gina Park, Danni Lu,  Mohammad Tarayani, Haiyan Deng, Oliver Gao
-   Institution: Cornell University Systems Engineering / Civil & Environmental Engineering

<hr></hr>

## Project Structure

This repository is organized into the following main directories:

- **[data/](https://github.com/gao-labs/nyc_congestion_pricing/tree/main/data)** - Contains data collection scripts, raw data files, and data processing utilities
- **[descriptives/](https://github.com/gao-labs/nyc_congestion_pricing/tree/main/descriptives)** - Contains descriptive analysis scripts and outputs
- **[plot/](https://github.com/gao-labs/nyc_congestion_pricing/tree/main/plot)** - Contains visualization scripts and generated plots

Each directory contains specialized scripts and data files relevant to different aspects of the NYC Congestion Pricing policy evaluation.
